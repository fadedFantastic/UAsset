Unity资源管理系统，资源加载底层主要使用xasset 2022的免费版框架，在此基础上借鉴开源库并扩展，主要用于个人项目及学习。

## 个人开发计划（咸鱼开发）:
1. 添加UAssetProxy入口类
2. 打包编辑面板调整
3. 添加示例场景
4. 运行时Debugger面板
